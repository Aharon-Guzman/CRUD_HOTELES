# CRUD Hoteles — Sistema de Gestión Hotelera

Aplicación web para la gestión de hoteles y sus datos administrativos. Desarrollada con JavaScript como proyecto académico en la Universidad Americana (UAM), enfocada en operaciones CRUD completas con validaciones y manejo de estado.

---

## Descripción

Sistema web que permite registrar, consultar, editar y eliminar información de hoteles. Proyecto orientado a practicar operaciones CRUD, manejo del DOM con JavaScript y consumo de datos de forma dinámica en el frontend.

---

## Stack Tecnológico

- **Frontend:** JavaScript ES6, HTML5, CSS3
- **Lógica:** JavaScript puro — sin frameworks adicionales
- **Interacción:** Manipulación dinámica del DOM

---

## Funcionalidades

- Listado de hoteles con visualización en tabla dinámica
- Registro de nuevo hotel con validación de campos
- Edición de datos de hotel existente
- Eliminación con confirmación
- Filtros de búsqueda por nombre o estado

---

## Cómo ejecutar

1. Clonar el repositorio:
```bash
git clone https://github.com/Aharon-Guzman/CRUD_HOTELES.git
```
2. Abrir `index.html` en el navegador — no requiere instalación adicional.
